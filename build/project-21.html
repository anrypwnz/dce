Проекты

Руководство модернизацией вычислительной инфраструктуры под ГИС

Заказчик

РТК Цифровые технологии

Москва, 2020

Для миграции информационных систем в составе одной из ГИС поставлена задача создания единой государственной облачной платформы, которая должна решить следующие задачи:

  • повышение доступности информационных систем;
  • повышение надежности хранения данных;
  • минимизация риска потери данных.
Решение

Обследована существующая инфраструктура ЦОД заказчика, предложены различные варианты развития, выбраны технические решения для повышения доступности информационных систем. Выполнен авторский надзор за выполняемыми работами.

Оборудование:

  • хранения данных – дисковые массивы Huawei семейства OceanStor;
  • вычислительное – стоечные серверы Huawei;
  • сети передачи данных – оборудование Huawei семейства CloudEngine;
  • сети хранения данных – оборудование Lenovo;
  • резервное копирование – оборудование HPE MSL.

Программное обеспечение:

  • виртуализация – программное обеспечение VMware;
  • резервное копирование – Backup and Replication производства Veeam.
Результат

Выполнено техно-рабочее проектирование, разработан комплект документации в составе 12-ти документов.
