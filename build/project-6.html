Проекты

Развитие корпоративной вычислительной сети и IP-телефонии в комплексе административных зданий

Заказчик

Водная компания

Москва, 2017

В рамках работ над проектом были решены следующие задачи:

  • обеспечение доступа к корпоративной телефонной связи для всех абонентов комплекса из шести административных зданий заказчика;
  • обеспечение вариативности для соединения абонентов с операторами, предоставляющими услуги городской, междугородной и международной связи;
  • замена устаревшего оборудования телефонной связи в комплексе административных зданий;
  • замена устаревшего оборудования корпоративной вычислительной сети в комплексе административных зданий;
  • обеспечение резервирования маршрутизации трафика Ip телефонии;
  • обеспечение стыковки IP телефонии комплекса административных зданий с подсистемами коллабораций других подразделений заказчика;
  • обеспечение единой системы мониторинга состояния корпоративной сети и сети Ip-телефонии;
  • расширение функциональных возможностей Ip телефонии.
Решение

В рамках работ была выполнена поставка оборудования, модернизация существующей локальной вычислительной сети, структурированной кабельной сети, телефонии, а также разработана рабочая документация на систему.

Оборудование:

  • телефонная станция – Cisco CUCM;
  • голосовой шлюз Cisco VG320-EM;
  • IP-Телефоны - Cisco CP-3905, Cisco CP-8865, Cisco CP-8865NR, Cisco CP-8865-W;
  • пульт диспетчерский – MARCONI;
  • программное обеспечение расширения функционала телефонной станции – Aurus;
  • коммутаторы ЛВС – Cisco Catalyst;
  • вычислительные системы - Cisco UCS.
Результат

Единая корпоративная система IP-телефонии, отвечающая текущим и перспективным потребностям заказчика.
