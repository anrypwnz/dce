Проекты

Модернизация корпоративной вычислительной сети

Заказчик

Фонд социального страхования Российской Федерации

Москва, 2019

В рамках концепции внедрений новых программных сервисов у Заказчика возникла необходимость в модернизации локальной вычислительной сети здания. Целями модернизации являлись:

  • увеличение общей производительности сетевой инфраструктуры за счет перехода на магистральные соединения 10 Гбит/с;
  • повышение уровня отказоустойчивости сетевой инфраструктуры;
  • подготовка инфраструктуры к внедрению сервисов, использующих технологию PoE;
  • внедрение решений по обеспечению безопасного доступа пользователей в сеть Интернет;
  • замена устаревшего, снятого с производства оборудования.
Решение

Была произведена поставка оборудования, подготовлена необходимая документация, выполнено внедрение решения.

Оборудование:

  • уровень доступа - стеки коммутаторов Cisco Catalyst 9300;
  • компонент защиты и контроля WEB-трафика - Cisco Web Security Appliance S395;
  • компонент управления защитой контента - Cisco Security Management Appliance M195.
Результат

В результате работ по модернизации была создана универсальная, производительная и отказоустойчивая сетевая инфраструктура, соответствующая всем требованиям для дальнейшего развития информатизации производственного процесса Заказчика.
