Проекты

Разработка рабочей документации для создания вычислительной инфраструктуры ЦОД

Заказчик

ФАУ «Главгосэкспертиза России»

Москва, 2018

Разработан техно-рабочий проект на создание вычислительной инфраструктуры ЦОД. Спроектированная инфраструктура решала следующие задачи:

  • увеличение вычислительных мощностей информационной инфраструктуры;
  • увеличение ёмкости и производительности ресурсов хранения данных;
  • модернизация сети передачи данных;
  • масштабирование сети хранения данных;
  • обеспечение резервирования и восстановления корпоративных данных.
Решение

Разработан комплект документации согласно комплексу стандартов ГОСТ 34. В состав разработанной документации входят:

  • пояснительная записка к техно-рабочему проекту;
  • спецификация оборудования;
  • план расположения оборудования и проводок;
  • программа и методика испытаний.
Результат

Сформирован комплект документации на создание вычислительной инфраструктуры ЦОД.
