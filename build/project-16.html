Проекты

Техническая поддержка резервного центра обработки данных

Заказчик

Водная компания

Москва, 2014 – 2016

В 2014-м году был построен резервный центр обработки данных, обеспечивающий катастрофоустойчивость информационных систем заказчика. Ввиду технологической новизны реализованных решений (использование архитектуры active-active с интеграцией виртуальной среды), Заказчик ежегодно выбирает подрядчика на основе открытого конкурса для оказания услуг технической поддержки.

Решение

Поддержка выполняется в режиме 9х5 со временем устранения неисправности на следующий рабочий день.

Оборудование:

  • хранения данных – семейств VNX, VPLEX, DataDomain производства EMC;
  • вычислительное – семейств PowerEdge производства Dell;
  • сети передачи данных – семейств Nexus, Catalyst производства Cisco Systems;
  • сети хранения данных – семейства 6xx производства Brocade;
  • спектрального уплотнения каналов – семейства FSP производства ADVA.

Программное обеспечение:

  • виртуализация – семейства ESX производства VMware;
  • операционные системы – семейства Windows Server производства Microsoft;
  • резервное копирование – семейства NetBackup Utility производства Symantec, семейства Avamar производства EMC;
  • кластеризация – семейства HA, DRS производства VMware.
Результат

Услуги по поддержке оказываются в срок и без замечаний.
