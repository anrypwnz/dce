Проекты

Управление проектом построения информационно-технологической инфраструктуры РЦОД и КСИБ

Заказчик

ПАО «Татфондбанк»

Казань, 2014 – 2016

Задача

Для повышения отказоустойчивости информационных систем Банка и удовлетворения требованиям информационной безопасности было принято решение организовать резервный центр обработки данных и комплексную систему информационной безопасности. Ввиду отсутствия экспертизы по реализации подобных проектов внутри Банка, был проведён закрытый конкурс по выбору подрядчика для управления проектом. По итогам конкурса была выбрана наша компания благодаря представленному опыту и широкому спектру компетенций сотрудников.

Решение

Для реализации проекта был организован ряд конкурсов по выбору подрядчиков, по итогам которых проектировщиками были выбраны компании ЗАО «Инфосистемы Джет» и ЗАО «Ай-Теко».

Оборудование:

  • хранения данных — семейств VPLEX производства EMC; MSA производства HP;
  • вычислительное — семейств BL производства HP;
  • сети передачи данных — семейств Nexus, ASR, Catalyst производства Cisco Systems;
  • сети хранения данных — семейства 6xx производства Brocade;
  • спектрального уплотнения каналов — семейства FSP производства ADVA.

Программное обеспечение:

  • виртуализация — семейства ESX производства VMware;
  • операционные системы — семейства Windows Server производства Microsoft;
  • резервное копирование — семейства NetBackup Utility производства Symantec;
  • кластеризация — семейства HA производства VMware, AlwaysOn производства Microsoft.
Результат

В 2014-м году был выполнен анализ бизнес-систем (BIA) и разработаны требования к параметрам RTO, RPO. В 2015-м году было выполнено проектирование технических решений ИТИ РЦОД и КСИБ. В 2016-м году запущен первый этап реализации.
