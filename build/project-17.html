Проекты

Проектирование и внедрение распределённого центра обработки данных

Заказчик

Медицинский страховой фонд

Москва, 2014 – 2015

Для снижения риска останова систем и потери данных фонда принято решение построить распределённый центр обработки данных. Работы выполнялись в два этапа - в 2014-м г. выполнено проектирование, в 2015-м г. осуществлены работы по запуску системы. Выбор исполнителя проводился на основании открытых конкурсных процедур.

Решение

Был спроектирован РЦОД с распределённым хранением данных в режиме active-active с распределённой СУБД Oracle средствами RAC (Real Application Cluster).

Оборудование:

  • хранения данных – семейств V-MAX, VNX, VPLEX производства EMC; Storwize производства IBM;
  • вычислительное – семейств Power, BladeCenter производства IBM;
  • сети передачи данных – семейств Nexus, ISR, Catalyst производства Cisco Systems;
  • сети хранения данных – семейства 6xx производства Brocade;
  • спектрального уплотнения каналов – семейства FSP производства ADVA.

Программное обеспечение:

  • виртуализация – семейства PowerVM производства IBM;
  • операционные системы – семейства AIX производства IBM;
  • резервное копирование – семейства Avamar производства EMC;
  • кластеризация – семейства Real Application Cluster производства Oracle.
Результат

Проектирование и внедрение проектных решений было успешно выполнено в 2014-м и 2015-м годах, соответственно.
