Проекты

Разработка рабочей документации и авторский надзор в рамках проекта по миграции информационной инфраструктуры банка

Заказчик

ЗАО «Америабанк»

Ереван, Республика Армения, 2017

Задача

В рамках программы повышения доступности информационных систем, банк осуществил консолидацию вычислительных мощностей из арендованного центра обработки данных и собственного ЦОД, имеющего ограничения к масштабированию, в новый собственный ЦОД, отвечающей перспективным потребностям банка.

Решение

Был разработан комплект рабочей документации, включающей план миграции информационных систем, схемы размещения оборудования в целевом ЦОД и схемы коммутации. Также, был осуществлен авторский надзор за выполняемыми работами.

Оборудование:

  • хранения данных — семейств 3PAR производства HPE; Storwize производства IBM;
  • вычислительное — семейства BladeCenter производства IBM; DL производства HPE;
  • сети передачи данных — семейств ASA, Catalyst производства Cisco Systems;

Программное обеспечение:

  • виртуализация — семейства Hyper-V производства Microsoft;
  • операционные системы — семейства Windows Server производства Microsoft;
Результат

Работы по миграции проведены в срок, согласно разработанному плану, ограничение предоставления ИТ-услуг было сведено к минимум и возникало только в рамках предусмотренных планом окон.
